packages/generator/src/cli/html-generator.ts: Generating the adventure website

The HTMLAdventureGenerator is the core orchestrator, responsible for managing the process of converting AI-generated narrative content into finished HTML pages. It embodies sequence control and encapsulates the complex pipeline to ensure modularity and clarity. Functions of interest include:

  • generateAdventure: Orchestrates the entire adventure generation process, from initializing the theme to producing final HTML files.
  • generateAllThemes: Configures a multi-theme workflow, ensuring shared assets are distributed once and that each theme generates in isolation.
  • startHttpServer: Launches a static HTTP server, enabling developers to preview the generated site directly.

Highlights

  • generateAdventure: Controls the sequential pipeline for generating a complete adventure site within a specific theme.
  • generateAllThemes: Facilitates the generation of multiple themes by isolating shared assets and using parallelized workflows.
  • startHttpServer: Allows instant preview by handling secure file serving and opening the final output in the user's default browser.

private async generateAllThemes(args: Map<string, string>): Promise<void> {
  console.log(chalk.green('‚úÖ Generating all themes'));
  const themes: AdventureTheme[] = ['space', 'mythical', 'ancient', 'developer'];
  const themePromises = themes.map(async theme => {
    const themeGenerator = new HTMLAdventureGenerator();
    themeGenerator['selectedTheme'] = theme;
    await themeGenerator.generateAdventure();
  });
  await Promise.allSettled(themePromises);
}
  • Implements efficient multi-threaded execution of isolated theme pipelines.
  • Ensures asset duplication is minimized by copying shared resources into global paths only once.
  • Promotes scalable extensibility by allowing additional themes to be introduced seamlessly.
private async startHttpServer(port: number = 8080): Promise<void> {
  const server = http.createServer((req, res) => {
    const filePath = path.join(this.outputDir, req.url || '');
    fs.readFile(filePath, (err, content) => {
      if (err) {
        res.writeHead(404);
        res.end('File not found');
      } else {
        res.writeHead(200);
        res.end(content);
      }
    });
  });
  server.listen(port, () => {
    console.log(chalk.green(`üåê HTTP server started on http://localhost:${port}`));
  });
}
  • Sets up a lightweight HTTP server to host generated sites locally.
  • Provides instant feedback by directly rendering output files in the browser.
  • Acts as a developer aid for debugging layout or runtime issues.
