Adventure Quests

🌠 Quest 1: Command Deck Protocols

You explore the nerve center of Starship Polaris by traversing src/server.ts and src/tools/tools.ts, witnessing how MCP tool interfaces coordinate your journey and how each adventure module is brought online for galactic exploration.

🚦 Quest 2: Astro-Quest Engine

You venture into the heart of the ship’s adventure system by analyzing src/adventure/adventure-manager.ts and src/adventure/story-generator.ts, discovering how the quest generation core orchestrates storylines and tracks your progress through the nebula.

🛰️ Quest 3: Nebula Scan Systems

You investigate the ship’s sensor array via src/analyzer/repo-analyzer.ts and src/llm/llm-client.ts, learning how Repomix scans the code nebula and how the LLM engines decode transmissions for stellar navigation.

🪐 Quest 4: Orbital Configurations

You configure the mission’s orbital parameters using src/shared/config.ts, src/shared/theme.ts, and src/shared/input-validator.ts, tailoring the starship’s adventure path and theme alignment for optimum navigation.
