Operation Starlight: Mapping the Galactic Code Nexus

In an interstellar age, the Federation's star archives face an enigmatic collapse—a surge of decentralized chaos threatening the stability of their galactic knowledge networks. To combat this, the Starship Prometheus Array has been outfitted with the Experimental Code Nexus (ECN), a machine built to decode rogue signal pathways and devise solution protocols. Prometheus Array relies on the synergy of its onboard systems: MCP tools for navigation, LLM interfaces for intel synthesis, and adventure-oriented Repomix modules for encrypted exploration. Your mission is to command the ship, uncover logical anomalies, and harmonize the structure of these scattered repositories into a cohesive galactic beacon. Each step takes you deeper into the Nexus, where fragments of code hold secrets to the Federation's survival.
