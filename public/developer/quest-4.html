packages/core/src/shared/theme.ts: Theme Definitions and Utilities

This file is a single source of truth for defining and managing themes across the adventure system. It includes predefined themes, such as "Space Exploration" and "Enchanted Kingdom," alongside utility functions for validating, parsing, and formatting themes. Each theme is crafted as a ThemeDefinition, containing values like emoji, description, and keywords for matching user input.

Highlights

  • THEMES contains predefined themes with attributes and descriptions.
  • getAllThemes() provides a utility to fetch all theme definitions in a logical array.
  • isValidTheme() determines if a given input matches any of the defined theme keys.
